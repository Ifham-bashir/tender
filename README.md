# Tender

&gt; **Gentle on you**

Tender is a skincare web experience that listens before it speaks. It builds a routine around how you feel, not just what you see.

## Philosophy

- **We don't fix skin. We accompany it.**
- No medical claims. No "7-day transformations." No pressure.
- Every recommendation explains *why* in human language.
- You can pause anytime. You can skip anything. You can say "I'm not sure."

## Features

| Feature | Description |
|---------|-------------|
| **Mood Check-in** | How you feel shapes your skin's needs. We ask first. |
| **Skin Profile** | Fitzpatrick type, age, concerns, budget, climate. Skip anything. |
| **Gentle Day Toggle** | A minimal routine mode for when life is heavy. |
| **Facial Sensor** | Optional photo analysis. Your photo never leaves your device. |
| **Pause Routine** | Your routine waits here. No guilt. No streaks. |
| **"Not Sure" Escape** | Every step has an alternative or a skip option. |
